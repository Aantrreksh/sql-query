Developing and running applications that use the Geospatial Toolkit

The Geospatial Toolkit provides a fully SQL/MM-compatible API for Spark SQL. Applications can use this API to analyze spatial data.

The examples in these topics illustrate how to manipulate and analyze spatial data:
  • Point locations, such as customers, retail stores, electrical transformers, and cell towers
  • Lines, such as highways, coastlines, delivery routes, and electrical transmission lines
  • Polygons, such as sales or service territories, flood or fire risk areas, states, provinces, and counties
The examples and the sample data are based on a tutorial that was created for Spatial Extender.
